AniList with custom video background + UI tweaks

This is a Chrome extension for AniList that lets you customize the website's background. It replaces the default static background with an animated video. The extension comes with a preset video from the anime The Elusive Samurai but also allows you to set any video you want by pasting a URL into the extension's popup window.

The extension also includes some UI tweaks by loading additional CSS files from another GitHub repository (carloshdrp/Anilist-CSS) and other stuff. These tweaks adjust the look of the navigation bar, footer, and other elements to complement the new video background.
